Le projet est issue d'un projet déja existant et commencé ultérieurement. Nous avons donc décidé d'utiliser ce cours pour mieux structurer notre projet.

Nom du jeu: Blade and Blossom

Explication global du jeu:
Il s'agit d'un jeu dans lequel le joueur parcours plusieurs plusieurs donjon, effectue des quêtes et améliore son personnage.

Résumé global du projet:
Dans ce projet (génie logiciel) nous avons décider de ne travailler que sur une partie du jeu qui correspond à la place central du jeu dans lequel le joueur peut se reposer entre ses missions, le VILLAGE.
Dans ce village le joueur pourra intérragir avec différents PNJ (personnages non jouable), différentes boutiques ainsi que différents animaux.
